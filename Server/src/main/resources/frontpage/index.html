Sobre

Criado a partir de uma ideia, este projeto pretende providenciar as instituições de solidariedade
com uma plataforma de catalogação de doações que permita ter:

  • Doações mais próximas às necessidades reais
  • Uma visão do andamento das campanhas de recolha em tempo real
  • Maior transparência e confiança para o publico
  • Maior participação das pessoas
  • Facilidade logística e inventariação
  • Incentivo à melhoria constante


Tecnologias

Criado com as mais recentes tecnologias, simplista e preparado para o futuro.

Desempenho

Tirando partido dos mais recentes padrões de desenho de software e bases de dados distribuídas, é possível obter alto debito de dados.

Disponibilidade

Preparado para trabalhar de forma distribuída e redundante, a plataforma providencia altas taxas de disponibilidade.

Open Source

Construído para servir de ideia para outros projetos e ajudar para um bom global, toda a gente pode contribuir.

Parcerias

ISEC

Instituto Superior de Engenharia de Coimbra, responsável pela lecionamento do Mestrado na qual este projeto se encontra inserido. É também um dos provedores de serviços da plataforma.

DEI-FCTUC

Departamento de Engenharia Informática, outro importante centro de ensino e igualmente provedor de serviços da plataforma.
